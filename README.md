## Description
This is a classic "Rock, Paper, Scissors" game where the player competes against the computer.

## How to Play

- Choose your move (Rock, Paper, or Scissors).
- Determine the winner based on the game rules:
  - Rock beats Scissors.
  - Scissors beats Paper.
  - Paper beats Rock.
